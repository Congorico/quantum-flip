# quantum-flip
O computador quântico usa a porta Hadamard (H) para colocar a moeda em um estado de 50\% cara e 50\% coroa ao mesmo tempo (superposição). Não importa se o humano vira ou não a moeda, a segunda jogada do computador quântico desfaz a superposição perfeitamente, garantindo que ela caia sempre em "Cara"
